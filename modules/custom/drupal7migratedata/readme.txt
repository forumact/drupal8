* drupal7Migratedata which helps to migrate following data from SqlBase.
   1. Drupal7 user table to Drupal8 User table
   2. Drupal7 user roles table to Drupal8 User Roles table
   3. Drupal7 node table to Drupal8 Node table
   4. Drupal7 Node Revision table to Drupal8 Node Revision table
   5. Drupal7 Comment table to Drupal8 Comment table
   6. Drupal7 Menu table to Drupal8 Menu table
   7. Drupal7 MenuLink table to Drupal8 MenuLink table
   8. Drupal7 Urlalias table to Drupal8 Urlalias table
   9. Drupal7 Domain table to Drupal8 Domain table
